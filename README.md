# GTAO 통계 뷰어 사용 설명서

## 프로그램 소개
GTA Online 플레이어의 통계를 조회하고 머니핵, 부캐릭터 사용 등 의심 지표를 분석해주는 프로그램입니다.

---

## 최초 설치 (처음 한 번만)

### 1. Chrome 확장 프로그램 설치
프로그램은 소셜클럽 데이터를 가져오기 위해 Chrome 확장 프로그램이 필요합니다.

1. Chrome 주소창에 `chrome://extensions` 입력
2. 우측 상단 **"개발자 모드"** 켜기
3. **"압축해제된 확장 프로그램 로드"** 클릭
4. 프로그램 폴더 안의 `gtao_extension` 폴더 선택

### 2. 소셜클럽 로그인
- https://socialclub.rockstargames.com 에서 Rockstar 계정으로 로그인합니다.
- **로그인 상태를 항상 유지**해야 프로그램이 정상 작동합니다.

---

## 프로그램 실행

### 파일 구성
프로그램을 실행하려면 아래 파일들이 같은 폴더에 있어야 합니다.
```
📁 폴더
├── GTAO통계뷰어.exe       ← 실행 파일
└── gtao_extension\
    ├── manifest.json
    └── background.js
```

### 실행 방법
`GTAO통계뷰어.exe`를 실행합니다.

---

## 셋업 화면

처음 실행하면 셋업 화면이 표시됩니다.

**STEP 1 — Chrome 확장 프로그램 설치**
위의 설치 과정에서 이미 완료했다면 넘어가도 됩니다.

**STEP 2 — 소셜클럽 로그인**
- **"소셜클럽 로그인 페이지 열기"** 버튼을 클릭하면 소셜클럽 페이지가 열립니다.
- 이미 로그인되어 있으면 자동으로 연결됩니다.
- 화면 하단에 ⏳ 아이콘이 표시되며 연결을 기다립니다.
- 연결되면 자동으로 설정 화면으로 이동합니다.

## 설정 화면

처음 연결 시 한 번만 표시됩니다. (이후엔 저장된 설정이 자동 적용)

| 항목 | 설명 | 기본값 |
|------|------|--------|
| 수익 오차 허용값 | 총수익-총지출과 실제 잔고의 차이가 이 값을 넘으면 머니핵 의심 | 1,000만 달러 |
| 시간당 수익 기준 | 시간당 이 금액 이상 벌었으면 머니핵 의심 | 200만 달러/시간 |

설정 완료 후 **"저장하고 시작하기"** 버튼을 클릭합니다.

> 설정은 나중에 우측 상단 **⚙ 판정 기준** 버튼으로 언제든지 변경할 수 있습니다.

## 검색 화면

### 기본 사용법
1. 닉네임 입력창에 조회할 플레이어의 **소셜클럽 닉네임**을 입력합니다.
2. **Enhanced / Legacy** 버튼으로 플랫폼을 선택합니다.
   - **Enhanced**: GTA Online 인핸스드 버전 (PC 최신버전)
   - **Legacy**: GTA Online 레거시 버전 (PC 구버전)
3. **"검색"** 버튼을 클릭하거나 Enter를 누릅니다.

> 닉네임은 최소 6글자 이상이어야 합니다.

### 상단 버튼
| 버튼 | 기능 |
|------|------|
| ⚙ 판정 기준 | 머니핵 의심 판정 기준값 변경 |
| 🔄 재설정 | 검색 결과를 지우고 처음 상태로 |

## 결과 화면

### 닉네임 행
- 검색한 플레이어의 닉네임이 표시됩니다.
- 우측의 **"🌐 통계 사이트"** 버튼을 클릭하면 해당 플레이어의 소셜클럽 통계 페이지로 이동합니다.

### ⚠ 의심 판정
| 표시 | 의미 |
|------|------|
| ✅ 의심 지표 없음 | 이상 지표가 발견되지 않음 |
| 🔴 수익 오차 $XXX — 머니핵 의심 | 총수익-총지출과 실제 잔고 차이가 허용값 초과 |
| 🔴 시간당 수익 $XXX — 머니핵 의심 | 전체 플레이타임 대비 수익이 기준값 초과 |
| 🔴 플레이타임 차이 XXX시간 — 부캐릭터 의심 | 전체 플레이타임과 캐릭터 플레이타임 차이가 24시간 초과 |

### 표시 항목 (왼쪽)
- **👤 기본 정보**: 랭크, 잔고
- **🏴 크루**: 크루 상징, 크루 이름(클릭 시 크루 페이지 이동), 색상, 태그
- **🔍 수익 오차 분석**: 총수익-총지출, 실제 잔고, 오차, 허용 오차
- **🚔 범죄 기록**: 경찰 치치, 수배 별 획득, 수배 상태 시간
- **💪 스킬**: 기력, 사격, 힘, 은신, 비행, 운전, 폐활량, 사이코 수치 (■■■□□ 형식)

### 표시 항목 (오른쪽)
- **📊 경력 통계**: 전체/캐릭터 플레이타임, KD 비율, 플레이어 사살/사망, 플레이타임 차이
- **💰 수익 내역**: 작업, 공유, 기타, 베팅, 차량 판매, 주운 금액, 총 수익/지출
- **⚔️ 경쟁 통계**: 레이스, 데스매치, 낙하산, 다트, 테니스, 골프 승/패

### 🏆 어워드
- 카테고리별 달성한 어워드 수가 표시됩니다. (예: `15 / 33`)
- 카테고리를 클릭하면 세부 어워드 목록이 펼쳐집니다.
- **초록색**: 달성 완료 / **빨간색**: 미달성 (현재값/목표값 표시)
- 어워드 항목에 마우스를 올리면 달성 조건 툴팁이 표시됩니다.

## 자주 묻는 질문

**Q. "확장 프로그램 연결 대기 중"에서 넘어가지 않아요.**
A. Chrome이 실행 중인지, 소셜클럽에 로그인되어 있는지 확인하세요. 소셜클럽 탭을 새로고침하면 자동으로 연결됩니다.

**Q. "존재하지 않는 닉네임입니다."가 표시돼요.**
A. 닉네임이 정확한지 확인하세요. 대소문자를 구분하지 않으며 철자를 정확히 입력해야 합니다.

**Q. 통계가 일부만 표시돼요.**
A. 해당 플레이어가 통계를 비공개로 설정했을 수 있습니다.

**Q. Enhanced/Legacy 중 어떤 걸 선택해야 하나요?**
A. 모르면 **Enhanced**로 검색하고, 결과가 안 나오면 **Legacy**로 바꿔서 검색해보세요.

**Q. 백신이 바이러스로 탐지해요.**
A. PyInstaller로 만든 exe 파일은 백신이 오탐하는 경우가 있습니다. 예외 처리 후 사용하세요.
