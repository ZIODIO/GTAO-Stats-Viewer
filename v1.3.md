## 변경 사항
- 어워드 항목을 보상 항목으로 변경
- 완료되지 않은 보상 항목을 흰색으로 표시하도록 변경
- 
##  버그 수정
- 보상 설명 순서 오류 수정

## 기능 추가
- 토큰 캐싱 (4분간 재사용, 검색 속도 향상)
- 닉네임 캐싱 (5분간 저장, 같은 닉네임 재검색 시 즉시 표시)
- 캐시된 결과면 의심 판정 위에 저장된 결과 (n초/분 전) 표시
- 보상 수치 조작 의심 판정 추가
- 보상 탭에서 수치 조작 의심 카테고리 항목을 빨간색으로 표시
